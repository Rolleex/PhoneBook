# PhoneBook
 Тестовое задание
 
Код состоит из двух частей:
- Класса PhoneBook 
- Основной функцией main.

#### В классе реализованно 6 функций

- load_phonebook

- save_phonebook

- display_page

- add_record

- edit_record
  
- search_records
  ____
### load_phonebook
 Открываем json файл и записывает его в список класса contacts, где у нас будут храниться все считанные данные.
____
 ### save_phonebook
 Открываем файл с разрешением для записи.
Сохраняем файл в json формате таким образом, чтобы не нарушить кодировку.
____

### display_page
Вывод постранично записей из справочника на экран
Мы задаем количество записей на странице:
```
records_per_page = 5
```
Далее вычисляем нужные индексы запсей.
```
start = (page_number - 1) * records_per_page
```
Где start - начальный индекс отображение записей
page_number = вводимое значение
Так как индексирование начинается с нуля, мы отнимаем 1 и умножаем на количество всех записей, для получение число записей в контактной книге.
Таким образом у нас есть необходимая нам точка отсчета для нужной пользователю страницы.
 end = start + records_per_page
 end - конечный индекс
 Добавляем к старту нужное количество записей.
 Далее перебираем каждую запись по заданным индексам в нашем списке
```
  for contact in self.contacts[start:end]:
```
и выводим красивым образом:

### add_record

Пользователь поочередной вводит необходимые данные.
Мы создаем новую запись под наш формат и присваиваем вводимые пользователем значения.
Добавляем запись в список класса и вызываем метод save_phonebook
____
### edit_record
Для редактирование записи нам потребуется индекс этой записи.
Делаем проверку на наличие данной записи.
Достаем заданнуб запись и присваиваем новые значения по каждому ключу.
Вызываем метод save_phonebook
____
### search_records
Создаем новый список.
Проверяем каждую запись
```
if any(search.lower() in value.lower() for value in contact.values()):
```
search - запрос поиска. 
Мы приводим запрос и значения к нижнему регистру и проверяем совпадения в значениях записей списка класса.
Если есть True - добавляем запись к нашему списку.
Возвращаем список.
____
## main 
```
 phonebook = Phonebook('phonebook.json')
```

Передаем нашему классу имя файла.

Запускаем цикл, для постоянной работы программы. 

При вызове каждой опции вызываем необходимый метод класса Phonebook.

Так, как при поиске мы создаем отдельный список,то тут же мы и реализуем вывод.
